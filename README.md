# Voltila Battery Alarm & Monitor

**Master Your Battery Life**

Voltila is an intelligent, customizable battery monitor for Windows designed for performance and aesthetics. Optimize charging, prevent overcharging, and extend your battery lifespan with style.

![Voltila Banner](https://i.postimg.cc/65KtqGd9/voltila-batterymonitor.jpg)

## Features

* **Smart Thresholds**
    Set custom notification limits. Choose between "Charge Limiter" or "Drain Alert" modes instantly to keep your battery healthy.

* **Audio Alerts**
    Never miss a notification with looping custom sounds. Includes built-in volume control for alerts.

* **Modern UI**
    Beautiful Glassmorphism design that adapts to your system with auto-switching Dark and Light themes.

* **System Tray Integration**
    Runs quietly in the background with negligible resource usage. Access settings with one click from your system tray.

1.  Download the latest installer from the link above.
2.  Run `Voltila_Installer_v1.0.0.exe`.
